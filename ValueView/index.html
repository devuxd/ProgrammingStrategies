Value View Demo

This is a simple demo of the value view. The parsing is pretty basic at this point. If the string is "nothing" or a link, it'll be formatted appropriately. If it's a comma separated list of more than one item, it'll be formatted as a list. There's a lot more complexity we can add to this later as necessary, but I think we should identify usability issues as we go.
